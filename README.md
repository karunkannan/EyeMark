# EyeMark
Simplify measurements for ultrasound images.

## Installation
1. Clone the repo
2. Run `pip3 install -r req.txt` to ensure that you have all the needed
   libraries.
3. Import the image folder into the directory
4. Run `python3 bin/eyemark.py`
